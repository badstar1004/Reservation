# Reservation
매장 예약 관리 프로젝트 (학습용 프로젝트)

# 기술 & 툴
- Java jdk 11
- MySql
- Spring Boot 2.7.10, gradle
- Web, JAP, Lombok, Validation, jwt, jackson-datatype, commons-lang3
- IntelliJ Ultimate

# 개발 기간
- 2023.04.12 ~ 2023.04.24

# 주요 기능
### 공통
- [x] 회원가입 (매니저, 일반회원)

### 매니저
- [x] 로그인 토큰 발행
- [x] 로그인 토큰을 통한 제어 확인 (JWT, Filter를 사용)
- [x] 매장 등록
- [x] 매장 예약 승인 / 예약 취소 등록

### 일반회원
- [x] 로그인 토큰 발행
- [x] 로그인 토큰을 통한 제어 확인 (JWT, Filter를 사용)
- [x] 매장 예약
- [x] 매장 리뷰 등록
